# Как добавить свою игру

Это короткая инструкция для тех, кто впервые видит проект. Игра здесь — обычная
небольшая страница на HTML, CSS и JavaScript. Node.js, игровой движок и отдельный
сервер не нужны.

## 1. Получите проект и запустите хост

```bash
git clone https://github.com/talgatv/no-signal-lan-arcade.git
cd no-signal-lan-arcade/pc
./start.sh
```

В Windows запустите `pc/start.bat`. Затем откройте в браузере
**http://127.0.0.1:8080/** и оставьте хост работающим в этом окне терминала.

## 2. Создайте заготовку

Откройте второе окно терминала, перейдите в корень репозитория и выполните:

```bash
python3 tools/new_game.py moi-shariki --title "Мои шарики" --author "Ваше имя"
```

Для сетевой игры добавьте `--multiplayer`:

```bash
python3 tools/new_game.py moi-shariki --multiplayer --title "Мои шарики" --author "Ваше имя"
```

Параметр `--author` необязателен: укажите его, если хотите публичное авторство.
Оба поля автора можно оставить пустыми. Идентификатор `ogh-team` предназначен
только для игр основной команды проекта.

Скрипт создаст:

```text
games/moi-shariki/
├── manifest.json
├── README.md
└── client/
    ├── index.html
    ├── style.css
    └── game.js
```

Он также добавит игру в `games/catalog/games.json`.

## 3. Сделайте игру

- `client/index.html` — экран и кнопки;
- `client/style.css` — внешний вид;
- `client/game.js` — правила и управление;
- `manifest.json` — название, число игроков и способы управления;
- `README.md` — короткое описание для проверяющего.

Откройте игру через хост:

**http://127.0.0.1:8080/games/moi-shariki/client/**

Не открывайте `index.html` через `file://`: модули и общие файлы так могут не
работать.

## 4. Соблюдайте основные правила

- Игра должна полностью работать без интернета: без CDN и обязательных облачных API.
- Размер папки игры — не более **10 МБ**, желательно меньше 2 МБ.
- Управление должно быть понятным на телефоне; добавьте большие touch-кнопки.
- Используйте своё название, код и графику; библиотека по умолчанию семейная.
- `id` должен состоять из строчных латинских букв, цифр и дефисов и совпадать с
  именем папки.

Общие стили, звук, профиль и сетевой клиент находятся в `games/_shared/`.
Их использование необязательно.

## 5. Проверьте результат

```bash
python3 tools/validate_catalog.py
du -sh games/moi-shariki
```

В Windows размер папки можно посмотреть через «Свойства» в проводнике.

После этого:

1. Откройте игру на компьютере.
2. Откройте её на телефоне в той же Wi-Fi сети:
   `http://<IP-компьютера>:8080/games/moi-shariki/client/`.
   Адрес локальной сети хост показывает при запуске.
3. Для сетевой игры проверьте две вкладки с одинаковым названием комнаты.

## 6. Отправьте игру

Создайте отдельную ветку и pull request. В описании напишите:

- как играть;
- одиночная игра или сетевая;
- на каких браузерах и устройствах вы её проверили.

Перед отправкой пройдите [чек-лист](./CHECKLIST.md). Подробности находятся в
[полной английской инструкции](./ADD_A_GAME.md), а сетевой режим описан в
[руководстве по multiplayer](./ADD_MULTIPLAYER_GAME.md).

Спасибо, что помогаете делать офлайн-библиотеку больше.
