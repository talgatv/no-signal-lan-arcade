About this project

Offline Games Hub (No-Signal LAN Arcade) turns one computer into a local game server. Friends join from any phone browser on the same Wi‑Fi — no app install for guests, no accounts, no cloud. Play tiny games and utility programs even when the internet is gone.

One host serves the library and multiplayer rooms. Progress and your profile stay only in this browser (you can export them as a file). Anyone can contribute a new game pack to the open-source collection.

Origin. The idea took shape when the project author was left without power during the 2025 Los Angeles wildfires — phones still had charge, people were in the same room, but the grid and the net were gone. A LAN arcade you can run from a laptop and a hotspot is built for exactly that moment: blackouts, disasters, travel, and any night the signal fails.
