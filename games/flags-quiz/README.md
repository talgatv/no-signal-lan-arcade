# Flags & Capitals

**Solo world-geography quiz + reference atlas** covering **every country** — all
193 UN member states plus the 2 UN observer states (Holy See / Vatican City and
the State of Palestine) = **195 countries**. This "193 members + 2 observers"
definition is the standard "complete list of countries" convention used by most
educational atlases, and it is this game's exact scope (no partially-recognised
states, dependencies, or territories).

Two features share one dataset (`client/data/countries.json`):

## 1. Quiz

Three multiple-choice modes, 4 options each:

| Mode | Prompt | Pick |
|---|---|---|
| **Flag → Country** | a flag | the country |
| **Country → Capital** | a country name | its capital |
| **Flag → Capital** | a flag | the capital (hardest — combines the other two) |

- **Region-plausible distractors.** The 3 wrong options are drawn from the same
  sub-region as the answer where possible (then same region, then anywhere), so a
  round can't be won by elimination — the choices for "France" are other European
  countries, not random noise.
- **Rounds** are 15 questions; a results screen shows score, accuracy, and your
  best (persisted per mode via `OGHProfile`, `localStorage` only).
- Optional **region filter** restricts a round to one continent.

## 2. Atlas (reference)

A browsable, searchable info sheet for **every one of the 195 countries** — not a
curated subset. Search by name (in any language) or code, filter by continent,
and tap a country for its card: flag, name (in the current UI language), capital,
region, approximate population and area, official language(s), currency, and ISO
code.

## Flags — hand-coded SVG

Every flag is **inline SVG written by hand from geometric primitives** (stripe /
band generators, an n-pointed-star polygon, an even-odd crescent path, Nordic and
upright crosses, triangles, discs, sun-ray fans, a Union-Jack helper, etc.) — no
bundled image files and no external requests at runtime. Flag colour/layout
*specifications* were researched (public-domain descriptions of official state
symbols); the SVG markup here is re-implemented fresh from those specs, not copied
from any third-party SVG source.

The large majority of world flags are genuinely simple geometric compositions and
are reproduced faithfully. About **64 flags** carry an intricate central emblem,
coat of arms, or seal (e.g. Mexico, Sri Lanka, Bhutan, Egypt, Croatia, Portugal,
Brazil): these render the **correct field, colours, and layout** with the emblem
**simplified or omitted**, and every such entry is marked `"detail": "simplified"`
in the data and shown a small "emblem simplified" note in the Atlas, so the
simplification is transparent rather than silently wrong.

### Data notes

- **Country names** are provided in all six UN languages (`en`, `ru`, `zh`, `es`,
  `ar`, `fr`) and displayed in whichever matches the current UI language.
- **Capital names are English only** (a deliberate, documented fallback — the task
  prioritised full 6-language coverage of country *names* over capitals). The rest
  of the info card follows the UI language.
- Population/area/currency/languages are **approximate reference figures** —
  realistic and correct in order of magnitude, not a live feed.
- A few countries that share their political flag with an intricate state emblem
  use the **civil flag** (e.g. Peru, Bolivia) which is the plain field.

## i18n

UI chrome (buttons, prompts, labels, results, region names) is fully translated in
all six UN languages via `i18n.js` (`?lang=` query param → `navigator.language` →
`en`; remembered in `localStorage`). Arabic switches the whole UI to **RTL** — but
the flag graphics and the country grid are deliberately **not** mirrored (the grid
container is `dir="ltr"`; SVG flags have their own coordinate system).

## How to run

```bash
cd pc && ./start.sh
# http://127.0.0.1:8080/games/flags-quiz/client/
```

Any device on the LAN can open that URL — no install, no build step, no runtime
network calls.

## Files

```text
flags-quiz/
├── manifest.json
├── README.md
└── client/
    ├── index.html   ← header/nav, quiz (setup/play/results), atlas, detail overlay
    ├── style.css    ← neon theme; html-scrolls / body-visible override for scrolling views
    ├── app.js       ← quiz engine (modes, distractors, scoring) + atlas (search/filter/detail)
    ├── i18n.js      ← UN-6 UI-chrome strings (RTL-aware)
    └── data/
        └── countries.json   ← 195 countries: 6-language names, capital, region,
                                population, area, languages, currency, hand-coded flag SVG
```

`countries.json` was generated by a build script (kept in the project scratchpad,
not shipped) that assembles each entry and composes its flag from the shared SVG
helpers; the JSON is the shipped source of truth.
