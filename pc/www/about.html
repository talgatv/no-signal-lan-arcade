About the project

When the internet dies, the party doesn’t have to.

What it is

Offline Games Hub (No-Signal LAN Arcade) turns one computer into a local game server. Friends join from any phone browser on the same Wi‑Fi — no app install for guests, no accounts, no cloud.

The host serves a lobby, a library of tiny games and utility programs, and optional multiplayer rooms. Your profile and progress stay only in this browser (you can export them as a file). Anyone can contribute a new pack to the open-source collection.

How it works

  • One machine runs the PC host (pc/start.sh)
  • Everyone opens the host URL in a browser
  • Pick games, chat, or tools — even with no internet

Origin

How it started. The idea took shape when the project author was left without power during the 2025 Los Angeles wildfires. Phones still had charge; people were in the same room; the grid and the network were gone. A LAN arcade you can run from a laptop and a hotspot is built for exactly that moment: blackouts, disasters, travel, and any night the signal fails.

Open source

MIT licensed. Contribute games, programs, and docs on GitHub. Lightweight packs (≤ 10 MB each) and an offline-first design are core rules.
